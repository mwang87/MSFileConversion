###Installing required packages
Required windows installation are included in the To_Install folder, but you might have to do different things based on your windows version. Currently supports Windows 10 64-bit version. Will likely work on Windows 8 64-bit version as well. 

###Converting files
To convert all files, put raw files without folder structure inside Input_Folder.

Double click on convert_all.bat. Converted files will be placed in Output_Folder. The log output will be placed in log.txt.
